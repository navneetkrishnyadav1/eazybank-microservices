# 🚀 EazyBank Microservices

A cloud-native **Spring Boot microservices system** implementing core banking services like **Accounts, Loans, and Cards**.

---

## ⚙️ Tech Stack

* Java, Spring Boot
* Spring Cloud (Eureka, Config Server, API Gateway)
* Docker, Kubernetes
* Kafka / RabbitMQ
* OAuth2, Keycloak
* Prometheus, Grafana

---

## 📌 Features

* REST APIs with Swagger documentation
* Service discovery & API Gateway
* Centralized configuration
* Event-driven architecture
* Security & observability

---

## ▶️ Run

```bash
mvn spring-boot:run
```

Swagger:

```bash
http://localhost:8080/swagger-ui.html
```
